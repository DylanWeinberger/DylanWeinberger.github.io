Development Portfolio

Jorts

Jorts is a location based app that allows users to find others in their area to do activities with. From throwing a frisbee around to intense pick-up basketball and even rounds of quizzo... JORTS wants to help you get out of the house! Jorts was built using Ruby on rails, Geocoder, Gmaps4Rails, and the google maps API.

Fatcat Smuggling

FatCat Smuggling is an app built using Ruby on rails and has intricate relationships and associations. The animations are done using Javascript and Jquery. The next step for this project will be turning it into a game based on the old ti-83 calculator game drug wars.

Bowie Box and Framework from Mars

The Bowie Box is a fully responsive Jukebox built using SASS, bourbon, and pure javascript. It also comes with a seperate front end framework. This comes with buttons, fonts, colors, spacing, and other styling in a David Bowie theme. It is fully responsive and has a totally different design depending on whether you are on desktop or mobile.

Justice Junction

Justice Junction is a blog built on Ruby on Rails, CSS, Javascript, and CarrierWave. The original design and styling was done in Adobe Illustrator. The blog was designed for use by intergalactic bounty hunters wanting to brag about capturing their victims in carbonite.
